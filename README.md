# Hospital Managment System

Backend application to be developed using Spring Cloud and Spring boot technologies stack by applying and  implementing microservices design patterens 

### Modules
* hospital-service (in-progress)
* fee-service (in-progress)
* hmis-cloud-config-server (Configuration Pattern)

other modules yet to be added which will implement following patterns.

* Netflix Eureka Naming Server
* Netflix Zuul API gateway Server
* Zipkin distributed Tracing Server
